# Anomali Sensor Pada Tanaman Cabai

Project Laravel untuk monitoring dan deteksi anomali pada tanaman cabai berbasis IoT.

## Fitur
- Monitoring sensor
- Deteksi anomali
- Dashboard statistik
- Upload gambar tanaman
- Sistem rekomendasi

## Teknologi
- Laravel
- Tailwind CSS
- MySQL
- Chart.js

## Cara Menjalankan

```bash
composer install
php artisan migrate
php artisan serve
